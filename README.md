# Amolenaria Boutique - Catálogo Online e Indumentaria

Este es un proyecto desarrollado como solución técnica para un emprendimiento real de calzado e indumentaria. El objetivo es centralizar la oferta de productos y agilizar el proceso de venta a través de WhatsApp.

## Relevamiento y Necesidades
A través de un proceso de análisis con la dueña del negocio, se identificaron las siguientes necesidades:
* **Catálogo centralizado:** Evitar la publicación manual e individual de productos en redes.
* **Reducción de consultas repetitivas:** Automatizar la información de talles, medios de pago y zonas de envío.
* **Agilidad en el cierre de venta:** Que el cliente llegue al chat con el modelo y talle ya definidos.

## Funcionalidades
- **Visualización Dinámica:** Catálogo generado mediante JavaScript para facilitar actualizaciones de stock.
- **Categorización:** Separación por Calzado, Bikinis, Pijamas e Indumentaria.
- **Guía de Talles:** Tabla interactiva con advertencias de variación según el modelo.
- **Información Logística:** Detalle claro de métodos de pago (Efectivo/Transferencia/MP) y envíos (Moto/Retiro).
- **Call to Action:** Botón de WhatsApp integrado para contacto directo.

## Tecnologías utilizadas
- HTML
- CSS (Diseño responsivo y personalizado)
- JavaScript (Inyección dinámica de productos)
- Git (Control de versiones)
- GitHub (Gestión de repositorio y despliegue)
